# Scripts Folder Structure Guide

This document describes the purpose of each folder inside the `scripts` directory.

---

## battle-view

Contains all scripts related to the **battle scene**.

This includes:
- Battle logic and flow
- Turn handling
- Attacks, abilities, and effects
- Battle UI logic
- Any systems used exclusively during battles

---

## code-related

Contains scripts related to **coding mechanics** in the game.

This includes:
- Programming or coding-based gameplay systems
- Logic tied to solving code-related challenges
- Opening and closing the in-game text editor
- Interaction between the game and the editor

---

## overworld-view

Contains all scripts related to the **overworld scene**.

This includes:
- Player movement
- Creature movement and behavior
- World interactions
- Camera logic for the overworld
- Any non-battle gameplay mechanics

---

## creatures-data

Contains scripts and data objects related to **creature definitions and storage**.

This includes:
- Creature stats and attributes
- Data models and ScriptableObjects
- Creature configuration files
- Any persistent data describing creatures
